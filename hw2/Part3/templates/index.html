Module 2 – Template Matching (Homework 2)

This module demonstrates template matching from Homework 2. The goal is to locate a smaller template image inside a larger scene image.

My implementation does the following:

  • Converts the scene and template images to grayscale.
  • Slides the template over the scene and computes a similarity score at each location using normalized cross-correlation.
  • Finds the best match location and draws a bounding box around the detected template.
